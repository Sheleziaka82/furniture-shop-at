# Анализ структуры каталога Planetmöbel.de

## Основные категории

На сайте партнеров выделены 3 основные категории:

### 1. Badezimmer (Ванная комната)
Подкатегории:
- Badmöbel-Sets (Наборы мебели для ванной)
- Gästebad-Sets (Наборы для гостевой ванной)
- Hochschrank (Высокие шкафы)
- Spiegelschrank (Зеркальные шкафы)
- Waschtischunterschrank (Тумбы под раковину)
- Badzubehör (Аксессуары для ванной)

### 2. Wohnzimmer (Гостиная)
Подкатегории:
- TV-Möbel (ТВ-мебель)
- Wohnzimmermöbel (Мебель для гостиной)

### 3. Garderobe & Flur (Прихожая и коридор)
Подкатегории:
- Garderoben-Sets (Наборы для прихожей)
- Schuhschränke (Обувные шкафы)
- Flurmöbel (Мебель для коридора)

## Дополнительные разделы
- Konfigurieren (Конфигуратор) - кастомизация мебели
- Sale (Распродажа)

## Ключевые характеристики товаров

### Для мебели ванной:
- Размер раковины (55 см, 60 см, 80 см и т.д.)
- Материал и цвет (Matt Weiß, Anthrazit, Holz Rustikal)
- LED-подсветка
- Конфигурация (количество дверей, ящиков)
- Стиль (Modern, Rustikal)

### Для мебели прихожей:
- Размеры (ширина, высота)
- Наличие зеркала
- LED-подсветка
- Количество отделений/клапанов
- Цвет (Weiß, Schwarz, Gold Eiche)

### Для ТВ-мебели:
- Размеры
- Цвет и материал
- Количество отделений для хранения

## Стилистика и особенности

- Фокус на **современном дизайне** (grifflose Türen - безручковые двери)
- Акцент на **функциональности** и **хранении**
- **Модульные системы** и конфигураторы
- **Немецкие названия** и описания
- Упор на **качество** и **быструю доставку**
- **30 дней** на возврат
