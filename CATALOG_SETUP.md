## Загруженные изображения

Все 13 изображений товаров успешно загружены в S3 хранилище и доступны по HTTPS ссылкам.

## Как добавить товары в базу данных

### Вариант 1: Через админ-панель (Рекомендуется)

1. Войдите в админ-панель как администратор
2. Перейдите в раздел "Produkte" → "Hinzufügen"
3. Скопируйте данные из вывода скрипта `seed-catalog-simple.mjs`
4. Заполните форму и добавьте URL изображения
5. Нажмите "Produkt hinzufügen"

### Вариант 2: Массовое добавление через SQL

Если нужно добавить все товары сразу, можно использовать SQL-запросы через Database UI в Management панели.

## Характеристики товаров

Каждый товар включает:
- **Название** (на немецком языке)
- **Описание** (детальное, на немецком)
- **Цена** (в евро)
- **Категория и подкатегория**
- **Размеры** (Ш x Г x В в см)
- **Материал** (MDF, Керамика, Стекло, Металл и т.д.)
- **Доступные цвета** (массив вариантов)
- **Количество на складе**
- **Изображение** (высокое качество, загружено в S3)

## Расширение каталога

Для добавления новых товаров:
1. Подготовьте изображения (рекомендуется 800x800px или больше)
2. Загрузите через форму в админ-панели
3. Заполните все поля (название, описание, цена, размеры, материал)
4. Укажите категорию и подкатегорию
5. Добавьте варианты цветов если есть

## Примечания
