# Руководство администратора - Möbelhaus AT

## Вход в админку

### Способ 1: Через веб-интерфейс
1. Откройте сайт вашего магазина
2. Нажмите кнопку **"Мein Konto"** (Мой аккаунт) в правом верхнем углу
3. Войдите с учетной записью владельца проекта
4. После входа нажмите **"Мein Konto"** снова
5. Вы увидите ссылку на **Admin Dashboard** (если у вас роль администратора)

### Способ 2: Прямой доступ
Перейдите по URL: `https://your-domain.com/admin`

## Требования для доступа

- Учетная запись должна иметь роль **`admin`** в базе данных
- Роль `admin` автоматически присваивается владельцу проекта
- Для других пользователей роль можно изменить в таблице `users` БД

## Основные разделы админки

### 1. Dashboard (Главная панель)
**Что здесь:**
- Ключевые метрики магазина
- Количество заказов за период
- Общая выручка
- Количество клиентов
- Последние заказы
- Популярные продукты

**Что можно делать:**
- Просматривать общую статистику
- Видеть последние активности
- Быстро переходить в другие разделы

### 2. Products (Управление продуктами)

#### Просмотр продуктов
- Список всех товаров в магазине
- Фильтрация по категориям
- Поиск по названию
- Сортировка по цене, популярности, дате добавления

#### Добавление нового продукта
1. Нажмите кнопку **"Добавить продукт"**
2. Заполните следующие поля:
   - **Название** - название товара
   - **Описание** - подробное описание
   - **Категория** - выберите категорию
   - **Цена** - цена в EUR
   - **Скидка** - процент скидки (опционально)
   - **Материал** - материал изготовления
   - **Цвет** - цвет товара
   - **Абмесения** - ширина, высота, глубина в см
   - **Вес** - вес в кг
   - **Количество на складе** - доступное количество
   - **Изображения** - загрузите фото товара

3. Нажмите **"Сохранить"**

#### Редактирование продукта
1. Нажмите на товар в списке
2. Измените необходимые данные
3. Нажмите **"Сохранить изменения"**

#### Удаление продукта
1. Выберите товар
2. Нажмите кнопку **"Удалить"**
3. Подтвердите удаление

### 3. Orders (Управление заказами)

#### Просмотр заказов
- Список всех заказов с датой и статусом
- Фильтрация по статусу (новый, обработан, отправлен, доставлен)
- Поиск по номеру заказа или имени клиента

#### Статусы заказов
- **Новый** - только что поступил
- **Обработан** - готов к отправке
- **Отправлен** - находится в пути
- **Доставлен** - получен клиентом
- **Отменен** - отменен по запросу

#### Обработка заказа
1. Нажмите на заказ
2. Просмотрите детали:
   - Информация о клиенте
   - Адрес доставки
   - Список товаров
   - Сумма и способ оплаты
3. Измените статус заказа при необходимости
4. Добавьте номер отслеживания для отправки
5. Нажмите **"Сохранить"**

#### Отправка уведомлений
- Система автоматически отправляет email-уведомления при изменении статуса
- Вы можете отправить дополнительное уведомление клиенту

### 4. Customers (Клиенты)

#### Просмотр клиентов
- Список всех зарегистрированных клиентов
- Информация о контактах
- Дата регистрации
- История заказов

#### Информация о клиенте
1. Нажмите на клиента
2. Просмотрите:
   - Личные данные
   - Адреса доставки
   - Историю заказов
   - Сохраненные товары (wishlist)
   - Баланс бонусных точек

#### Управление клиентом
- Можно отредактировать контактные данные
- Можно добавить примечание
- Можно просмотреть все его заказы

### 5. Categories (Категории)

#### Просмотр категорий
- Список всех категорий товаров
- Количество товаров в каждой категории

#### Добавление категории
1. Нажмите **"Добавить категорию"**
2. Введите название категории
3. Добавьте описание (опционально)
4. Нажмите **"Сохранить"**

#### Редактирование категории
1. Нажмите на категорию
2. Измените данные
3. Нажмите **"Сохранить"**

### 6. Settings (Настройки)

#### Общие настройки
- Название магазина
- Описание
- Контактная информация
- Адрес магазина
- Телефон
- Email

#### Настройки доставки
- Способы доставки
- Стоимость доставки
- Сроки доставки
- Регионы доставки

#### Настройки платежей
- Способы оплаты
- Комиссии
- Валюта

#### Email-уведомления
- Включить/отключить автоматические уведомления
- Шаблоны писем
- Email для получения уведомлений

- **Регулярно обновляйте количество** - следите за наличием товаров
- **Добавляйте качественные фото** - хорошие изображения увеличивают продажи
- **Используйте описания** - подробное описание помогает клиентам принять решение

- **Отправляйте уведомления** - информируйте клиентов о статусе заказа
- **Добавляйте номер отслеживания** - позволяет клиентам отслеживать доставку

### Аналитика
- **Отслеживайте популярные товары** - какие товары продаются лучше всего

## Часто встречающиеся проблемы

### Не вижу админ-панель
**Решение:**
1. Убедитесь, что вы вошли с правильной учетной записью (владельца проекта)
2. Проверьте роль в базе данных: `SELECT * FROM users WHERE openId='your-id';`
3. Если роль не `admin`, обновите: `UPDATE users SET role='admin' WHERE openId='your-id';`

### Не могу добавить товар
**Решение:**
1. Проверьте, что все обязательные поля заполнены
2. Убедитесь, что изображение загружено корректно
3. Проверьте размер файла (не более 5MB)
4. Попробуйте обновить страницу и повторить

### Заказ не отображается
**Решение:**
1. Проверьте фильтры - может быть установлен фильтр по статусу
2. Попробуйте поиск по номеру заказа
3. Проверьте дату - может быть установлен фильтр по периоду

### Не отправляются email-уведомления
**Решение:**
1. Проверьте настройки email в разделе Settings
2. Убедитесь, что email-адрес клиента верный
3. Проверьте логи сервера на ошибки
4. Попробуйте отправить тестовое письмо

## Безопасность

### Рекомендации
- **Используйте сильный пароль** - минимум 12 символов с буквами, цифрами, спецсимволами
- **Не делитесь учетными данными** - только вы должны иметь доступ к админке
- **Регулярно проверяйте логи** - смотрите, кто и когда заходил в админку
- **Делайте резервные копии** - регулярно сохраняйте данные
- **Обновляйте систему** - устанавливайте обновления безопасности

### Двухфакторная аутентификация (2FA)
- В разработке - будет добавлена в ближайшее время
- Обеспечит дополнительный уровень защиты
